# inner constant ION_CONST

Html attribute name constants

mostly straightforward with the notable exceptions:

  • ION_CONST.error.append variable append strategy - que root of the css query
  • ION_CONST.error.queries:

    • ION_CONST.error.queries.query the media query that while be made via HTMLElement#querySelectorAll

    • ION_CONST.error.queries.variables variables that will be set/unset: the keys will be concatenated with '--' eg: key => element.style.setProperty('--' + key, variables[key].set)

      The placeholder ${name} can be used to mark the field's name
